# 简介

---

**MarkDown**： 是一种轻量级标记语言， 在2004年由约翰·格鲁伯（英语：John Gruber）创建，生成的文件后缀是.md或.markdown。

它的应用很广泛，可以用于支持：

- 支持在GitHub、Gitee、GitLab等平台上用于编写Wiki、ReadMe等文档说明
- 支持在GitBook上撰写电子书
- 支持导出PDF、WORD、Html等不同格式文档[^1]

**MarkDown** 很强大，语法支持也很多有：

- 标题、文本格式、列表、图片、链接、目录[^2]

- 代码块、注释、类图

- 数学公式

- 甘特图、UML图、Mermaid流程图、Flow流程图、饼状图

  ```mermaid
  graph TB
      A[把大象放进去] --> B{"冰箱小不小？"}
      B -->|不小| C[把冰箱门关上]
      B -->|小| D[换个大冰箱]
  ```
