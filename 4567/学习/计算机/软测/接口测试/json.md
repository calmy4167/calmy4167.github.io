# Json[^1]

**介绍**：轻量级数据交换格式，存储和交换文本信息的语法。

**优点**

- 易于人阅读和编写
- 易于机器解析和生成

**格式**

- 对象

  ```javascript
  {
    "name": "John Doe",
    "age": 30,
    "isEmployee": true
  }
  ```

- 数组

  ```json
  [
    {
      "name": "Apple",
      "color": "Red"
    },
    {
      "name": "Banana",
      "color": "Yellow"
    }
  ]
  ```
