java数组

初始化

```java
//静态初始化
int[] arr1={1,2};
int[][] arr2={
    {1,2},
    {3,4}
};
//动态初始化
int[] arr3=new int[3];
int[][] arr4=new int [6][7];
//---------------
int[] arr1=new int[4];
int[][] arr2=new int[3][];
int[] arr3={1,2,3};
int[][] arr4= {
		{1,2},
		{3,4}
};
int[][] arr5=new int[][] {
	{1,2},
	{3,4}
};
```

Java中的this

- 使用范围：只能在构造函数和实例方法中使用，不能再静态方法中使用
- 用法：
  1. 引用类变量
  2. 在构造方法中调用同一个类的另一个构造方法
